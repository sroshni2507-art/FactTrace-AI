# FactTrace-AI
FactTrace-AI: Truth Verification & Mitigation System
FactTrace-AI is a high-impact, end-to-end Machine Learning solution designed to combat the global challenge of Fake News. It not only detects misinformation using advanced NLP techniques but also features a unique Automated Mitigation System to trace and correct the spread of false information at the source.
🚀 Project Overview
In today’s digital world, fake news spreads faster than the truth. FactTrace-AI solves this problem by:
Detection: Analyzing news text using a trained Machine Learning model.
Verification: Comparing input against linguistic patterns of verified news.
Tracing: Identifying emotional bias and key terms that drive misinformation.
Mitigation: Directly alerting potential spreaders via SMS/Alerts with official truth links.
✨ Key Features
AI-Powered Detection: Built with a Passive Aggressive Classifier for high-speed, high-accuracy text classification.
Sentiment & Bias Analysis: Traces whether the news is informative or emotionally manipulated (Biased).
Interactive Dashboard: A clean, professional UI built with Streamlit (inspired by FakeShield AI).
Word Cloud Visualization: Automatically generates a visual map of the most impactful keywords in the news.
Automated Mitigation (Unique): Integration capability with SMS APIs (like Twilio) to send "Truth Alerts" to spreaders.
Real-time Credibility Score: Provides a percentage-based confidence score for every verification.
🛠️ Tech Stack
Language: Python 3.x
Framework: Streamlit (Web Dashboard)
Machine Learning: Scikit-learn (Passive Aggressive Classifier, TF-IDF Vectorizer)
Natural Language Processing (NLP): NLTK, TextBlob
Data Visualization: Matplotlib, WordCloud
Deployment: Streamlit Cloud / GitHub
